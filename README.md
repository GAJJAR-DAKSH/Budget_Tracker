# 💰 Budget Tracker Pro

A modern and responsive Budget Tracker built with **HTML, CSS, and JavaScript** to help users manage income, expenses, and financial insights efficiently.

## 🚀 Features

### 📊 Dashboard Analytics

* Total Income
* Total Expenses
* Current Balance
* Monthly Analytics

### 🥧 Expense Visualization

* Interactive Pie Chart using Chart.js
* Category-wise expense distribution

### 💳 Transaction Management

* Add transactions
* Delete transactions
* Categorize expenses and income
* Search transactions instantly

### 🌙 Dark / Light Mode

* Theme toggle support
* User preference saved locally

### 📄 Reports & Export

* Export transactions as CSV
* Download detailed PDF reports

### ⚡ Recent Activity

* View latest transactions
* Quick financial overview

### 💾 Local Storage

* Data persists even after browser refresh
* No backend required

### 📱 Responsive Design

* Desktop friendly
* Tablet friendly
* Mobile friendly

## 🛠️ Technologies Used

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/GAJJAR-DAKSH/Budget_Tracker.git
```

Navigate to the project folder:

```bash
cd Budget_Tracker
```

Open `index.html` in your browser.
